单例模式

什么是单例？就是保证在一个jvm当中只能由一个实例。（区分不是在多个jvm当中。）

单例模式有七种写法。

本文只提出两种，一种懒汉式和饿汉式。

懒汉式

被需要时才进行初始化，且只有同一个实例，但会发生线程不安全。

饿汉式

这两者的区分：懒汉式不是线程安全的，且效率比饿汉式要低（因为加了同步），但是节约内存
。但饿汉式是天生线程安全的，因为当class被加载的时候就已经被初始化了。
