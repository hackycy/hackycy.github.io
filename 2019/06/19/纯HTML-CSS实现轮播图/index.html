图片并排显示

可以看到正常文档流图片是向下排列的，如何让他们实现并排显示呢，可能会想到使用float，但是最简单的办法是使用flex布局。给slide-wrap容器指定为Flex布局。

为什么slide-wrap要给予500%的宽度，因为该列有五张图片，一张图片刚好占满一个父容器。
